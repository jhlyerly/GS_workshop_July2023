Data Cleaning with R

These are the files associated with the Data Cleaning with R workshop session. In this folder there is an R project, a markdown file, folders for raw and processed, or clean, data, and a PDF of the slides.

Students should have R and RStudio installed prior to the workshop. 
The Appendix from Hands-On Programming with with R provides some instructions for beginners: https://rstudio-education.github.io/hopr/starting.html.

Getting started:
Open the R project - data_cleaning_with_R_project.Rproj
Open the markdown file - data_cleaning_with_R.Rmd

Folders:
data_raw:
Contains the raw data files for this workshop session

data_clean:
Contains the clean data files at the end of the session

Other files:
Slides: data_cleaning_with_R_slides.pdf
Markdown PDF: data_cleaning_with_R_Rmd.pdf
Markdown html: data_cleaning_with_R.html
